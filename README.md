Synapse: The Agentic Study Brain
Problem Being Solved: Preparing for massive, multi-stage exams over a 2.5-year timeline requires a living memory system. Traditional notes fail to account for the human forgetting curve or interdisciplinary connections. Synapse uses the Membrain API to actively track memory decay (surfacing topics before you forget them) and automatically builds a semantic graph linking related concepts across different subjects.

Tech Stack: Python, FastAPI, Membrain API
